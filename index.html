    Tic-Tac-Toe App

  • Game Board
    • The game would feature a 3x3 grid representing the Tic-Tac-Toe board. This could be implemented using a 2D array or a collection of objects representing each cell.
  • Player Turns
    • Game would alternate between two players, represented as "X" and "O." Each player would take turns selecting an empty cell on the board to place their marker.
  • Results
    • Win Condidition - win can occur if a player has three of their markers in a horizontal, vertical, or diagonal line. If a win condition is met, the game would declare the winner and end the game. Tie - If all cells on the board are filled and no player has won, the game would be declared a tie.
  • Restart
    • After a game ends, the players could have the option to restart and play another round without relaunching the program.
